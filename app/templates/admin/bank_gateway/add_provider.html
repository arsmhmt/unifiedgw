Provider Guidelines
What is a Bank Provider?

A bank provider is an intermediary account that facilitates transactions between clients and the banking system.

Commission Rates
  • Deposit Commission: Percentage charged on deposit transactions
  • Withdrawal Commission: Percentage charged on withdrawal transactions
  • Default is 0% - can be adjusted per transaction type
Transaction Limits
  • Set a maximum transaction amount per operation
  • Leave as 0 for unlimited transactions
  • Can be adjusted later based on provider performance
Provider Status
  • Active Provider can process transactions
  • Blocked Provider is temporarily disabled
  • Status can be toggled anytime from the provider list
Important: Ensure all information is accurate before creating the provider. The user account cannot be changed later.

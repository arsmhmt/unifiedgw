1. Create a USDT Deposit

Send a request to create a USDT deposit session for your user.

POST /api/v1/crypto/deposits
Authorization: Bearer <YOUR_API_KEY>
Content-Type: application/json

{
  "fiat_amount": 250,
  "fiat_currency": "USD",
  "crypto_currency": "USDT",
  "crypto_network": "TRC20"
}

Response example:

{
  "success": true,
  "payment_id": 1234,
  "transaction_id": "b1f8e...",
  "fiat_amount": 250.0,

  "network": "TRC20",
  "deposit_address": "TYASr5...",

2. Receive Webhook on Confirmation

When the deposit is confirmed on-chain, PayCrypt will call your webhook URL.

POST https://your-platform.com/webhooks/paycrypt/deposit

{
  "payment_id": 1234,
  "transaction_id": "b1f8e...",
  "amount": 249.1,
  "currency": "USDT",
  "network": "TRC20",
  "status": "confirmed",
  "txid": "blockchain_hash_abc123"
}

You update the user balance when status is confirmed.

Python Example

import requests

resp = requests.post(
  "https://paycrypt.online/api/v1/crypto/deposits",
  headers={"Authorization": "Bearer <API_KEY>"},
  json={
    "fiat_amount": 100,
    "fiat_currency": "USD",
    "crypto_currency": "USDT",
    "crypto_network": "TRC20"
  }
)
print(resp.json())

PHP Example

<?php
$ch = curl_init("https://paycrypt.online/api/v1/crypto/deposits");
$data = [
  "fiat_amount" => 100,
  "fiat_currency" => "USD",
  "crypto_currency" => "USDT",
  "crypto_network" => "TRC20"
];
curl_setopt_array($ch, [
  CURLOPT_POST => true,
  CURLOPT_HTTPHEADER => [
    "Authorization: Bearer <API_KEY>",
    "Content-Type: application/json"
  ],
  CURLOPT_POSTFIELDS => json_encode($data),
  CURLOPT_RETURNTRANSFER => true
]);
$response = curl_exec($ch);
curl_close($ch);
print($response);

Node.js Example

import fetch from 'node-fetch';

const res = await fetch('https://paycrypt.online/api/v1/crypto/deposits', {

  body: JSON.stringify({
    fiat_amount: 100,
    fiat_currency: 'USD',
    crypto_currency: 'USDT',
    crypto_network: 'TRC20'
  })
});
const data = await res.json();
console.log(data);

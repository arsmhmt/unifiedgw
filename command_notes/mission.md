# Mission

This repository is part of the wider KOMUTAN / PayCrypt ecosystem.

Purpose:
- Provide a unified crypto + bank payment gateway for betting and high-risk operators.
- Expose clear APIs and admin tools for clients, branches, and providers.
- Power higher-level products (risk tools, reporting, marketing funnels).

Success:
- Operators can plug in PayCrypt as a single gateway for crypto + bank rails.
- Demo + marketing flows (Engine A/B) consistently generate qualified leads.
- Platform remains maintainable as new flows and regions are added.
